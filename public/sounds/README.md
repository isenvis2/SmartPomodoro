# 알람 사운드 폴더

원하는 알람 소리를 mp3 파일로 만들어 이 폴더에 아래 이름으로 넣으면
포모도로 타이머가 자동으로 재생합니다. 파일이 없으면 기본 비프음이 재생됩니다.

## 1. 알람음 (전환 알림, 2종류)

세션이 끝나는 순간 짧게 재생됩니다.

| 파일명 | 재생 시점 |
|---|---|
| `focus-end.mp3` | 집중 세션 종료 → 휴식 시작 (작업의 모든 세션이 끝났을 때도 이 소리 재생) |
| `break-end.mp3` | 휴식 종료 → 다음 집중 시작 |

파일이 없으면 자동으로 비프음이 재생됩니다.
설정(⚙️) 화면에서 소리/진동/화면 깜빡임을 각각 켜고 끌 수 있고, "테스트" 버튼으로 미리 들어볼 수 있습니다.

## 2. 배경음악 (BGM, 2종류)

타이머가 진행되는 동안 계속 재생되는 음악입니다. 단계가 바뀌면 자동으로 곡이 전환됩니다.

| 파일명 | 재생 구간 |
|---|---|
| `focus-bgm.mp3` | 집중 시간 동안 재생 (집중에 어울리는 음악) |
| `break-bgm.mp3` | 짧은/긴 휴식 시간 동안 재생 (휴식에 어울리는 음악) |

설정(⚙️)에서 "세션 중 음악 재생"을 켜야 동작하며, 파일이 없으면 재생되지 않습니다.
